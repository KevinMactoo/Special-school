Code Club Kenya

Code Club Kenya is one of the main Projects Under iEARN. iEARN(International Education and Resource Network) is an organization in more than 140 countries. iEARN empowers educators and young people to work together online using the Internet and other new technologies. Over 2,000,000 young people each day are engaged in collaborative project work worldwide.
